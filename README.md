# ⚡ Max.dt88 — Dream State Edition

This is the high-aesthetic, interactive version of the Silent Strength website. It uses Next.js, Tailwind CSS, Framer Motion, and Three.js to create a premium experience.

## 🚀 Launch Sequence

To see this site live on your machine, follow these steps:

### 1. Install Node.js
If you don't have Node.js installed, download it from [nodejs.org](https://nodejs.org).

### 2. Enter the project folder
Open your terminal and type:
```bash
cd "/Users/maximedoucet/Documents/Anti gravity/1st Web/site"
```

### 3. Install Dependencies
```bash
npm install
```

### 4. Start the Engine
```bash
npm run dev
```

### 5. View the Masterpiece
Open your browser to: [http://localhost:3000](http://localhost:3000)
